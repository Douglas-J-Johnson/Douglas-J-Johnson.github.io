Projects


Journal Insights

Journal Insights is an application that allows a user to create and store journal entries and then get insights on the entries from IBM Watson Tone Analysis and Personality Insights services. The insights for an entry are displayed in a simple graphical format. The application is built on a Rails backend that uses a Postgres database, with a Javascript front end using IBM Watson functionality from both the front and back end.


Countries

The Countries application pulls country data from the restcountries.eu API and creates cards to display information about the the countries of the world. The user can perform various filters and sorts and display country population and area information on a world map. The application is built with a React fronend using the Plotly Javascript library for map functionality.


Puzzler

Puzzler creates puzzles from random, or random based on user specified keywords, Unsplash images. THe user can adjust the number of columns and rows to split the image into, and then solves the puzzle by dragging pieces. Puzzler is built on Rails back end with a React front end.
